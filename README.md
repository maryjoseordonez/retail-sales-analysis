# retail-sales-analysis
Retail Sales Data Analysis using Python, SQL and Data Visualization

This project explores retail sales data to identify key business insights related to revenue performance, product profitability, and customer behavior.

## Dataset

Superstore retail dataset containing sales transactions across multiple regions and product categories.

## Objectives

* Analyze overall revenue performance
* Identify top-performing products
* Explore regional sales trends
* Understand customer segments

## Tools Used

* Python
* Pandas
* SQL
* Data Visualization (Matplotlib)

## Key Insights

* Total revenue analyzed: $12.6M
* Central region generated the highest sales
* July showed one of the highest sales peaks
* Canon imageCLASS 2200 Advanced Copier was the most profitable product

## Project Files

* `sales_analysis.ipynb` → Python analysis notebook
* `superstore_clean.csv` → dataset used in the analysis
